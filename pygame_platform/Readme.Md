# Enkellt exempel på hur man kan hantera

* Tangentbordsinmatning
* Rörelse
* Kollisionsdetektering med rektanglar

En grund att bygga vidare sitt plattformspel med.

För att komma igång behöver man. 
* ha en hyfsad ny Python-version och Pygame installerat (Exemplet byggdes med Python Python 3.12.3 & Pygame 2.6.1
* Hämta katalogen och öppna den med valfri kodeditor (VSCode eller PyCharm rekomenderas)
* Det kan vara värt att skapa en virtuell miljö.
